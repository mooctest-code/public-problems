### 题目描述

查找字符串中在给定位置之间出现的所有数字，返回数字列表

### 输入描述

第一行测试用例的个数
下一行为测试用例的字符串
下一行为开始位置和结束位置，中间用空格隔开

### 输出描述

字符串中所有数字的列表

### 测试样例

#### 样例1: 输入-输出

```
1
iselab 123 mooc 456 huidhui
1 20
```

```
['123', '456']
```
