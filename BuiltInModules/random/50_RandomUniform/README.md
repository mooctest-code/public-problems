### 题目描述

请你生成一个 10 到 20 之间的随机数

### 输入描述

输入是一个整数 seed, 表示一个随机数种子

### 输出描述

请你以 seed 为随机数种子，生成一个 10 到 20 之间的随机浮点数，结果保留到三位小数

### 测试样例

#### 样例1: 输入-输出

```
2019
```

```
18.323
```

### 提示

`random.seed()` 和 `random.uniform()`
