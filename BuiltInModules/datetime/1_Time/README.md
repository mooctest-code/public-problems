### 题目描述

时间戳表示自从1970年1月1日午夜（历元）经过了多长时间。

请将一个时间戳格式化输出。

### 输入描述

输入为一个浮点数，表示时间戳。

### 输出描述

使用 `Year-Month-Day Hour:Minute` 的格式输出该时间戳表示的时间。

### 测试样例

#### 样例1: 输入-输出

```
1555134795.4250443
```

```
2019-04-13 13:53
```
