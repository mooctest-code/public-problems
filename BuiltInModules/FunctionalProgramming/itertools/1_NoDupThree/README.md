### 题目描述

给你四个数字，能组成多少个互不相同且无重复数字的三位数？请你输出每一个三位数。

### 输入描述

一行四个数字，每个数字的范围均是 1 \~ 9，两两之间用空格隔开，保证没有重复的数字。

### 输出描述

请你按照组成的三位数的大小，从小到大输出所有的三位数，每个占一行。

### 测试样例

#### 样例1: 输入-输出

```
1 2 3 4
```

```
123
124
132
...
431
432
```
