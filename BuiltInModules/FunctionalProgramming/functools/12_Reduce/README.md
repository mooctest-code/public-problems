### 题目描述

请使用 functools 中的 reduce 函数来计算一个列表的和。

### 输入描述

输入为一个列表 L。

### 输出描述

输出为 L 的元素和

### 测试样例

#### 样例1: 输入-输出

```
[1, 2, 3]
```

```
6
```

#### 样例2: 输入-输出

```
['a', 'b', 'c']
```

```
abc
```

### 说明

你需要使用 functools 中的 reduce 来完成这道题
