### 题目描述

给你三条边长，请你先判断其是否能构成三角形，若能，输出这个三角形的边长和面积，否则输出 `Can't construct a triangle.`。

### 输入描述

输入为三条边长，均为正整数，两两之间用空格隔开

### 输出描述

若该三条边能构成三角形，输出该三角形的周长和面积，面积保留两位小数，否则输出 `Can't construct a triangle.`

### 测试样例

#### 样例1: 输入-输出

```
4 5 3
```

```
12 6.00
```

#### 样例2: 输入-输出

```
3 4 7
```

```
Can't construct a triangle.
```
