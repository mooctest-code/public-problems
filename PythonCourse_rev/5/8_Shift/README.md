### 题目描述

给定正整数 n（1 < n < 100），验证命题：如果一个三位整数是 n 的倍数，则这个整数循环左移后得到的另两个三位数也是 n 的倍数。命题为真则输出“这是一个真命题”；命题为假则输出“这是一个假命题”。

### 输入描述

输入为一个整数 n

### 输出描述

若题目描述中的命题为真，输出 `这是一个真命题`，否则输出 `这是一个假命题`

### 测试样例

#### 样例1: 输入-输出

```
37
```

```
这是一个真命题
```

#### 样例2: 输入-输出

```
12
```

```
这是一个假命题
```
