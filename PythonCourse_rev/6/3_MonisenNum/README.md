### 题目描述

P 是素数且 M 也是素数，并且满足等式 M=2^P - 1，则称 M 为默尼森数。例如，P=5， M=2^P-1=31，5 和 31 都是素数，因此 31 是默尼森数。编写判断素数的函数并找出第 n 个默尼森数。

### 输入描述

输入为一个正整数 ｎ（0 < n < 7）

### 输出描述

输出第 n 个默尼森数

### 测试样例

#### 样例1: 输入-输出

```
2
```

```
31
```
