### 题目描述

请从具有 `My GPA is: x.` 模式的字符串中提取出 GPA 的值 x，结果为浮点类型。比如从 `My GPA is: 3.5.` 提取出 GPA 为 3.5。

### 输入描述

输入为一个模式为 'My GPA is: x.' 的字符串

### 输出描述

从该字符串中提取出 GPA x，x 为浮点数

### 测试样例

#### 样例1: 输入-输出

```
My GPA is: 3.5.
```

```
3.5
```

#### 样例2: 输入-输出

```
My GPA is: 4.0.
```

```
4.0
```
