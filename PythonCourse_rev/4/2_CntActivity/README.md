### 题目描述

现在有一个列表，记录了每次主题演讲活动的参与学生姓名。请根据这个列表统计出每个学生参加活动的次数，并记录到字典中。

### 输入描述

输入为一个记录每次主题演讲活动的参与学生姓名的列表，两两间用空格隔开

### 输出描述

输出每个学生的姓名和参加活动的次数，格式为 `姓名: 次数`，按学生姓名字典序排列，每个学生占一行

### 测试样例

#### 样例1: 输入-输出

```
xiaoma xiaowang xiaoma xiaoliu xiaoma xiaoliu
```

```
xiaoliu: 2
xiaoma: 3
xiaowang: 1
```
