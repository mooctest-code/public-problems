### 描述

Python可以作为一个功能强大的计算器，同时又不仅仅是一个计算器，Python还可以帮助我们进行字符串运算。操作以下给定的字符串，获取正确的结果

### 输入

一共四行，每行对应一个字符串a, b, c, d

### 输出

拼接字符串a和b，获得字符串c的第五个字符，重复输出十次字符串d

### 样例输入

```
Hello 
Python!
I Love Python
Hi
```

### 样例输出

```
Hello Python
v
HiHiHiHiHiHiHiHiHiHi
```
