### 题目描述

给你n个正整数，请你将它们组合成一个它们能组成的最大的正整数。比如两个整数 `123` 和 `13`，它们能组成的最大的整数是 `13123`。

### 输入描述

输入为一组正整数，两两间用空格隔开

### 输出描述

输出为它们能够组成的最大的正整数

### 测试样例

#### 样例1: 输入-输出

```
12 123 13
```

```
1312312
```

#### 样例2: 输入-输出

```
7 12 4 246
```

```
7424612
```

### 提示

使用 functools 中的 cmp_to_key 自定义排序
