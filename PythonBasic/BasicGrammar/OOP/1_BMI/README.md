### 题目描述

身体质量指数（BMI，Body Mass Index）是国际上常用的衡量人体肥胖程度和是否健康的重要标准，计算公式为：BMI=体重/身高的平方（国际单位kg/㎡）。

定义BMI类，将身高体重作为 `__init__()` 方法的参数，在 `__init__()` 方法中计算BMI指数，并使用 `printBMI()` 方法输出BMI指数（保留一位小数）。

### 输入描述

输入为两个浮点数表示某人的身高（m）和体重（kg）

### 输出描述

输出这个人的身体质量指数，保留一位小数

### 测试样例

#### 样例1: 输入-输出

```
1.70 62.4
```

```
BMI: 21.6
```
