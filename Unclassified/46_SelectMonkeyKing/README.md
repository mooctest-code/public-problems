### 题目描述

有ｎ只猴子，按顺时针方向围成一圈选大王（编号从１到ｎ），
从第１号开始报数，一直数到ｍ，数到ｍ的猴子退出圈外，剩下的猴子再接着从1开始报数。
就这样，直到圈内只剩下一只猴子时，这个猴子就是猴王，
现在给你n和m，请你输出最后猴王的编号。

### 输入描述

输入为猴子的数量 n 和报数 m

### 输出描述

输出为最后猴王的编号

### 测试样例

#### 样例1: 输入-输出

```
10 23
```

```
10
```
