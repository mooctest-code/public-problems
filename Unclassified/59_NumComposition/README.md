### 题目描述

有一个元素大小不超过 9 的整数数组，请问该数组中的整数能组成多少个互不相同且数字不重复的两位数？

### 输入描述

第一行为该数组的长度 n

第二行为 n 个不大于 9 的整数，两两间用空格隔开

### 输出描述

输出为互不相同且数字不重复的两位数的个数

### 测试样例

#### 样例1: 输入-输出

```
3
1 2 9
```

```
6
```

### 说明

组成的意思是 1, 2, 9 可以组成 12、19、29、21、91、92 这6个两位数
