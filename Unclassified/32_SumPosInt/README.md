### 题目描述

给你一个正整数 N，请你求出一共有多少种方式用不超过 N 的正整数求和得到 N。

### 输入描述

输入为一个正整数 N (0 < N < 100)

### 输出描述

输出为用不超过N的正整数求和得到N的方式种数

### 测试样例

#### 样例1: 输入-输出

```
4
```

```
5
```

### 提示

```
4 = 4
4 = 3 + 1
4 = 2 + 2
4 = 2 + 1 + 1
4 = 1 + 1 + 1 + 1
```
