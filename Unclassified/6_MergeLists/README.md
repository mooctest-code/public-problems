构造字典

### 题目描述

字典是一个键值对集合，比如 `{'a': 1, 'b': 2}` 中，`'a'` 是键，`1` 是值。给你两个具有相同长度的列表 a 和 b，请你以 a 中的元素为键，b 中对应位置的元素为值，构建一个字典。

### 输入格式

输入为三行，第一行是一个正整数 N。

第二行有 N 个字符串，两两之间用一个空格隔开，这些字符串组成列表 a。

第三行有 N 个整数，两两之间用一个空格隔开，这些整数组成列表 b。

### 输出格式

使用 N 行输出这个字典，每行格式为 `key: value` 。

### 测试样例

#### 样例1: 输入-输出

```
3
abc def ghi
1 2 3
```

```
abc: 1
def: 2
ghi: 3
```
