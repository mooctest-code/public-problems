### 题目描述

星期一到星期日的英文为Monday、Tuesday、Wednesday、Thursday、Friday、Saturday和Sunday，编写程序实现输入数字n（1~7之间）输出对应星期几的英文，程序无需考虑输入数字异常的情况。

### 输入描述

输入为一个数字表示一个星期的第 n 天，假设每周的第一天是星期一

### 输出描述

输出这一天的英文名。

### 测试样例

#### 样例1: 输入-输出

```
2
```

```
Tuesday
```
