阶乘除法

### 题目描述

输入两个正整数 n, m, 输出 n!/m!, 其中阶乘定义为 `n! = 1*2*3*...*n (n>=1)`。

比如,若 n=6, m=3,则 n!/m! = 6!/3! = 720/6 = 120。是不是很简单?

现在让我们把问题反过来: 输入 k = n!/m!, 请找到这样的整数二元组(n,m) (n>m>=1)。

### 输入描述

输入为一个整数 k (0 < k < 10^7)

### 输出描述

输出为一个元组 (n, m)，为满足 k = n!/m! 的一个整数二元组。 

如果答案不唯一, 请输出 n 最小的一个二元组。

### 测试样例

#### 样例1: 输入-输出

```
72
```

```
(9, 7)
```
