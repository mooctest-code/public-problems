实训三：逻辑运算

### 描述

Python可以提供强大的逻辑运算功能帮助我们来判断两个对象是否一致。帮助小南比较经过一番运算后的对象之间是否还相等。

### 输入

一共四行，每行对应一个字符串 a, b, c, d

### 输出

分别对a，b和c，d进行加法计算，判断运算后的值是否相等。

### 样例输入

```
Hello 
Python
Hello
o Python
```

```
'2'
'1'
'1'
'2'
```

### 样例输出

```
'Hello Python' == 'Hello Python' is True
```

```
'21' == '12' is False
```
