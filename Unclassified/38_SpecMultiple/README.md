### 题目描述

一个数的特殊倍数是指仅由1和0构成的倍数。如10是2的倍数，同时它只由1和0构成，因此它是2的特殊倍数。

### 输入描述

输入为一个正整数 n (0 < n < 10^3)

### 输出描述

输出为 n 的最小特殊倍数 m，我们保证 m 不超过整数范围

### 测试样例

#### 样例1: 输入-输出

```
2
```

```
10
```

#### 样例2: 输入-输出

```
3
```

```
111
```
