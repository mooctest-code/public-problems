### 描述

小南现在是班长了，需要组织大家参加运动会，小南现在需要把报名参加运动会的同学的名字记录下来，并且可以很快的获得任意一个参赛同学的名字，但是有没有合适的方法，请你帮帮小南。

### 输入

一共四行，每行对应一个人名字符串。

### 输出

第一行：使用list组织的学生名单l1；
第二行：输出l1中第三个学生的姓名；
第三行：获得l1的逆序list并输出；

### 样例输入

```
a
b
c
d
```

### 样例输出

```
['a', 'b', 'c', 'd']
c
['d', 'c', 'b', 'a']
```
