### 题目描述

斐波那契数列（Fibonacci sequence），又称黄金分割数列，指的是这样一个数列：`0, 1, 1, 2, 3, 5, 8, ...`  。

在数学上，费波那契数列是以递归的方法来定义：
$$
F_0=0,F_1=1\\F_n=F_{n-1}+F_{n-2}, n\geq 2
$$
请你使用上述公式计算得到第 n 个斐波那契数。

### 输入描述

一个整数 n (0 < n < 20)。

### 输出描述

输出第 n 个斐波那契数。

### 测试样例

#### 样例1: 输入-输出

```
3
```

```
1
```
