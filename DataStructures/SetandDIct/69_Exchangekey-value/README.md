### 题目描述

请将字典中的键和值交换并返回新的字典。

### 输入描述

输入为一个字典

### 输出描述

输出为一个键和值交换过的字典，字典按照键升序排列输出

### 测试样例

#### 样例1: 输入-输出

```
{'Wangbing': 1001, 'Maling': 1003, 'Xulei': 1004}
```

```
{1001: 'Wangbing', 1003: 'Maling', 1004: 'Xulei'}
```
