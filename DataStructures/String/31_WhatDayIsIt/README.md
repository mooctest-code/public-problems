### 题目描述

输入一周中某一日的英文名称的前n个字母，请你判断是星期几。

### 输入描述

输入是一个字符串 s，为一周中某一日的英文名称的前 n 个字母，大小写不定。

### 输出描述

请你判断字符串 s 代表的是星期几，首字母大写。

### 测试样例

#### 样例1: 输入-输出

```
su
```

```
Sunday
```

#### 样例2: 输入-输出

```
M
```

```
Monday
```
