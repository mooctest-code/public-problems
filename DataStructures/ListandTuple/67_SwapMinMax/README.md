### 题目描述

输入一个数组，请你交换其中的最大值和最小值，输出交换后的数组

### 输入描述

第一行为一个正整数 n 表示数组的长度

第二行为 n 个整数，两两间用空格隔开

### 输出描述

请你输出交换最大值和最小值后的数组，元素两两间用空格隔开

### 测试样例

#### 样例1: 输入-输出

```
5
1 2 3 4 1
```

```
4 2 3 1 4
```
