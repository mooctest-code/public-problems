批量修改字符串

### 题目描述

假设已有一个姓名列表，从输入获得一个表示姓名对应人共性内容的字符串和一条下划线，将其插入列表中每个姓名前，输出序号（从1开始）和修改后的姓名。

### 输入描述

输入分为两行，第一行为一个每个名字前需要加的共性内容字符串，第二行为姓名列表。

### 输出描述

输出除了需要加上共性前缀外，需要再在前面加一个序号，如样例输出所示

### 测试样例

#### 样例1: 输入-输出

```
NJU_
['Liuyun', 'Xuxu', 'Yangjun']
```

```
['1 NJU_Liuyun', '2 NJU_Xuxu', '3 NJU_Yangjun']
```
