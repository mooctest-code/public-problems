### 题目描述

请将输入的数字分别表示为二进制、八进制和十六进制

### 输入描述

输入为一个数字

### 输出描述

输出三个字符串，分别表示这个数字的二进制、八进制和十六进制，格式参考测试样例

### 测试样例

#### 样例1: 输入-输出

```
100
```

```
0b1100100 0o144 0x64
```

#### 样例2: 输入-输出

```
-100
```

```
-0b1100100 -0o144 -0x64
```
