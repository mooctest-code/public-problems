### 题目描述

现在有一个学生类，请你使用 setattr() 和 getattr() 补充代码（不允许修改该学生类），为该学生对象增加姓名、学号和年级，并输出相关信息

### 输入描述

输入为三个字符串 name, sno 和 grade，分别表示姓名、学号和年级

### 输出描述

输出这个学生对象的信息，参考测试样例

### 测试样例

#### 样例1: 输入-输出

```
小明 1632436 大三
```

```
姓名: 小明
学号: 1632436
年级: 大三
```
