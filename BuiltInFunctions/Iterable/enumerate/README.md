### 题目描述

对于输入的员工列表，我们默认从第一个员工开始为员工编号，第一个员工的初始编号为10001，输出所有员工的编号和员工姓名。

### 输入描述

为员工姓名的列表，姓名之间用空格隔开

### 输出描述

所有员工id和姓名，中间用空格隔开

### 测试样例

#### 样例1: 输入-输出

```
tony sunny louis 
```

```
10001 tony
10002 sunny
10003 louis
```
