整数表示

### 题目描述

在IA32上无符号的整型变量在内存中是“倒”存储的。
如一个整数: 305419896 换算成16进制是0x12345678，在内存中从低地址到高地址的4个字节里的数据分别是 0x78 0x56 0x34 0x12。

### 输入描述

输入一个整数 n (-10^10 < n < 10^10)

### 输出描述

输出其在内存中从低地址到高地址的存储数据（16进制请大写表示），两个数据之间用空格隔开。

### 测试样例

#### 样例1: 输入-输出

```
0
```

```
0x00 0x00 0x00 0x00
```

#### 样例2: 输入-输出

```
5987456
```

```
0x80 0x5C 0x5B 0x00
```

### 提示

需要考虑负数的情况，但不用考虑溢出
