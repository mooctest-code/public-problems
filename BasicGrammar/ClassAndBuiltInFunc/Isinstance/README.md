### 题目描述

使用python中的isinstance函数，判断读取的变量是否为int 或 float 类型

### 输入描述

可能为int类型或sring类型或float类型

### 输出描述

是否为int 或 float 类型，是则输出True，否则输出False

### 测试样例

#### 样例1: 输入-输出

```
2
```

```
True
```

#### 样例2: 输入-输出

```
'asd'
```

```
False
```
