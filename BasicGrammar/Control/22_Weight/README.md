### 题目描述

要想在一个天平上称出 1 到 n 克重的物品，至少需要多少个砝码？假设砝码可以放在两边。

### 输入描述

输入为最大需要称的重量 n (0 <= n <= 10^18)

### 输出描述

输出为称从 1 到 n 克至少需要的砝码数量

### 测试样例

#### 样例1: 输入-输出-解释

```
40
```

```
4
```

```
要称小于或等于 40g 的物品，至少需要四个砝码，分别重 1, 3, 9, 27 克。
```
