杨辉三角

### 题目描述

杨辉三角形是二项式系数的一种写法，形似三角形，在中国首现于南宋杨辉的《详解九章算术》得名，书中杨辉说明是引自贾宪的《释锁算术》，故又名贾宪三角形。前 5 行写出来如下：

```
1
1 1
1 2 1
1 3 3 1
1 4 6 4 1
```

杨辉三角形第 n 层正好对应于二项式 $(a+b)^(n-1)$ 展开式的系数。比如 $(a+b)^2=a^2+2ab+b^2$ 的系数为 `1 2 1`，对应杨辉三角第三层。

给你一个正整数 n，请你输出前 n 层的杨辉三角。

### 输入描述

一个正整数 n

### 输出描述

输出前 n 层的杨辉三角

### 测试样例

#### 样例1: 输入-输出

```
3
```

```
1
1 1
1 2 1
```
