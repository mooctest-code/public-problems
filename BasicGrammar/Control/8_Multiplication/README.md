### 题目描述

“一一得一，二二得四”，大家小时候都背过乘法口诀表吧，下面试着用程序来输出一个乘法口诀表。

### 输入描述

一个正整数 n

### 输出描述

输出 n*n 的乘法口诀表，格式参考样例输出，每行最后一个式子后没有空格。

### 测试样例

#### 样例1: 输入-输出

```
3
```

```
1*1=1
2*1=2 2*2=4
3*1=3 3*2=6 3*3=9
```
