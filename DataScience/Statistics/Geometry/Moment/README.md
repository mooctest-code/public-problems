### 题目描述

请你实现函数 `A(k, x)` 和 `B(k, x)` 分别计算一组数据 x 的 k 阶原点矩和 k 阶中心矩。

### 输入描述

输入为一组浮点数，两两间用空格隔开。

### 输出描述

输出这组数据的前 4 阶原点矩和中心矩。每阶占一行，结果均保留四位小数。

### 测试样例

#### 样例1: 输入-输出

```
7.6 2.6 3.2 4.4 5.4 6.8 8.8
```

```
5.5429 0.0000
35.2800 4.5567
246.8389 0.7722
1835.2827 34.2564
```

#### 样例2: 输入-输出

```
6.69 5.09 7.46 4.67 6.77 5.09
```

```
5.9617 0.0000
36.6443 1.1028
231.7433 0.1331
1503.1585 1.6144
```
