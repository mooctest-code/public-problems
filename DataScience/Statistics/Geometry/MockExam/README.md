### 题目描述

现有某班级学生本学期两次模拟考试的物理成绩，试求这两次模拟考的协方差和相关系数。

### 输入描述

输入两行。第一行为第一次模拟考的成绩；第二行为第二次模拟考的成绩。两两间用空格隔开。

### 输出描述

输出这两次模拟考的协方差和相关系数。

### 测试样例

#### 样例1: 输入-输出

```
78 74 60 42 53 63 70 90 54 82
85 69 76 53 70 59 82 95 60 67
```

```
128.7400 0.7398
```

#### 样例2: 输入-输出

```
60 42 53 68 70 90 54 82
59 82 95 60 87 70 53 70
```

```
-34.6250 -0.1681
```
