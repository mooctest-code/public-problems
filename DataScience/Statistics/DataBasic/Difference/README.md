### 题目描述

现有某校毕业生的就业年薪匿名调查表，请你分别求出该就业年薪表的偏差平方和、方差和标准差。

### 输入描述

输入为一个就业年薪列表，每一个值代表该毕业生的年薪（浮点数，单位为 w），两两间用空格隔开。

### 输出描述

输出年薪的偏差平方和、方差和标准差，用空格隔开，保留两位小数。

### 测试样例

#### 样例1: 输入-输出

```
7.6 2.6 3.2 4.4 5.4 6.8 8.8
```

```
31.90 4.56 2.13
```
