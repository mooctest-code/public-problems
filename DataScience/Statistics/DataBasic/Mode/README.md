### 题目描述

现有某校毕业生的就业年薪匿名调查表，请你将年薪从 0 开始、间隔为 3 进行分组，分成 \[0, 3\), \[3, 6\), \[6, 9\) …，求出人数最多的分组。

### 输入描述

输入为一个就业年薪列表，每一个值代表该毕业生的年薪（浮点数，单位为 w），两两间用空格隔开

### 输出描述

输出人数最多的分组，如果有多个，每个占一行，按年薪从小到大排列

### 测试样例

#### 样例1: 输入-输出

```
4.2
```

```
[3, 6)
```

#### 样例2: 输入-输出

```
7.6 2.6 3.2 4.4 5.4 6.8 8.8
```

```
[3, 6)
[6, 9)
```

### 提示

可以手动实现，也可以使用 `scipy.stats` 来解决。
