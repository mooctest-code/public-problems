### 题目描述

设有 n 台同类型的设备，各台工作是相互独立的，发生故障的概率都是 0.01。现在需要指派一些人共同维护这 n 台设备。试计算至少需要多少维修工人才能保证每台设备的及时维修率大于 0.99。

### 输入描述

输入为设备数 n。

### 输出描述

输出至少需要的维修工人数。

### 测试样例

#### 样例1: 输入-输出

```
80
```

```
3
```

#### 样例2: 输入-输出

```
20
```

```
2
```
