### 题目描述

某电子元件的寿命 X（小时） 服从参数为 λ=0.001 的指数分布，试求该元件能使用 n 个小时以上的概率。

### 输入描述

输入为使用时间 n。

### 输出描述

输出能够使用 n 个小时以上的概率，结果保留六位小数。

### 测试样例

#### 样例1: 输入-输出

```
100
```

```
0.904837
```

#### 样例2: 输入-输出

```
1000
```

```
0.367879
```
