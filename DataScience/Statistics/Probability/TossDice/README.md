### 题目描述

抛掷一枚质地均匀的骰子，请计算在随机数种子为 0 下，抛掷 n 次后，1~6 面每个面正面朝上的频率。

### 输入描述

输入为一个整数 n。

### 输出描述

输出抛了 n 次后，1 到 6 面每个面出现的频率，保留两位小数。

### 测试样例

#### 样例1: 输入-输出

```
10
```

```
0.10 0.00 0.30 0.50 0.10 0.00
```

#### 样例2: 输入-输出

```
999
```

```
0.19 0.15 0.17 0.17 0.15 0.17
```

### 注

请使用 `randint()`。
