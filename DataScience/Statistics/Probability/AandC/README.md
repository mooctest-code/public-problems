### 题目描述

请实现排列组合中的排列数公式 $A^m_n$ 和组合数公式 $C^m_n$ 。

### 输入描述

输入为两个非负整数 n 和 m，表示从 n 个元素中取出 m 个元素。

### 输出描述

输出 $A^m_n$ 和 $C^m_n$。

### 测试样例

#### 样例1: 输入-输出

```
4 2
```

```
12 6
```

#### 样例2: 输入-输出

```
10 3
```

```
720 120
```
