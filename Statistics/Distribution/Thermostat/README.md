---
题目: 温度调节器
简介: 求温度调节器定的温度
难度: 2
标签: 概率分布
作者: Unknown
慕码: Unknown
---

### 题目描述

将一温度调节器放置在贮存着某种液体的容器内，调节器温度定在 d 度，若液体的温度服从平均温度为温度调节器温度，标准差为 0.5 的正态分布，若要保持液体的温度至少为 c 度的概率不低于 p，试求 d 至少应为多少。

### 输入描述

输入为至少需要保持的最低温度 c 和概率 p。

### 输出描述

输出调节器的温度 d，精确到两位小数。

### 测试样例

#### 样例1: 输入-输出

```
80 0.99
```

```
81.16
```

#### 样例2: 输入-输出

```
37 0.95
```

```
37.82
```
