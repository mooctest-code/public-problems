简介: 有 n 个人射击靶子，求命中率
难度: 2

作者: 匿名

### 题目描述

有 n 个人同时向一个靶子射击一枪，已知他们每个人的命中率，求靶子被命中的概率。

### 输入描述

输入为 n 个人的命中率，两两间用空格隔开。

### 输出描述

输出靶子被命中的概率，结果保留四位小数。

### 测试样例

#### 样例1: 输入-输出

```
0.5 0.4 0.8 0.6
```

```
0.9760
```

#### 样例2: 输入-输出

```
0.3 0.4 0.2
```

```
0.6640
```
